MERN Boilerplate Server

The following are the endpoints of the api

Method Route Parameters Access Description
POST /api/auth/register body : JSON{name, email, password} Public A route for user to register
POST /api/auth/login body : JSON{email, password} Public A route for user to login
GET /api/profile/getprofile headers : Authorization : Bearer token Private A route for user to get user profile
PUT /api/profile/updateprofile
  • headers : Authorization : Bearer token
  • body : JSON{id, name, email}
 Private A route for user to update user profile
POST /api/auth/sendOTP body : JSON{email} Public A route for user to request for otp to recover password
PUT /api/auth/reset/:id body : JSON{password, id} Public A route for user to reset password
